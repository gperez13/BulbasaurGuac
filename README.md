# BulbasaurGuac


Phaser based space shooter featuring the original Pokemon starter underdog: Bulbasaur. 

In the future, additional elements should be added including power ups and animations. Bullets should also be redifined for smoother performance. 



1) Open Game On Browser
2) Follow Controlls to Shoot Mewtwo Horde
3) Avoid Moltres
4) Click on In-Screen Message When Round Is Over Or When Pokeballs = 0
